Навчальний модуль: Масиви: одновимірні та двовимірні

Цей ресурс містить навчальні матеріали з інформатики на тему "Масиви: одновимірні та двовимірні", розроблені для учнів 10-11 класів. Курс розрахований на 22 уроки та призначений для учнів, які не мають попередніх знань з даної теми.

Метою є формування базових знань та практичних навичок роботи з одновимірними та двовимірними масивами в програмуванні.

Огляд курсу

Для детального ознайомлення з курсом, його структурою, цілями та очікуваними результатами навчання, будь ласка, перегляньте файл README.md.

Структура матеріалів

  • Детальний план навчального модуля (Уроки, підтеми, ключові поняття, практичні завдання)
  • Матеріали уроків (Теоретичні матеріали, приклади коду, завдання)
  • Матеріали для оцінювання (Тестові питання, завдання для контрольних робіт)
  • Додаткова документація (Звіти, рефлексія, джерела)

Будь ласка, зверніть увагу, що деякі розділи можуть бути порожніми або містити заглушки, оскільки це базова структура. Матеріали будуть додаватися згідно з розробленим планом.
